pav::IGameObjectBase class

Base class for all game objects.

Contents

  • Reference
    • Derived classes
    • Constructors, destructors, conversion operators

This class is meant for game object usage. You should not derive your custom game object from this class. For custom game object deriving, refer to the class IGameObject
