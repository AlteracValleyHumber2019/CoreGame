pav::IScene class

Scene interface used to hold all game play related objects.

Contents

  • Reference
    • Public functions

The scene is the super container for almost all game play related object. It has an inner SEC system that managers all IGameObjectBase objects and IComponentBase objects.

class ExampleScene : public pav::IScene

public:
    // Inherited functions

    virtual void EndScene() override;


    virtual void Update(const float delta_time) override;


    virtual void SetupEngineEvents(pav::EventAttorney* event_attorney) override;

    // Custom function used for a keydown event handle
    void OnKeydown(pav::KeyCode keycode);

};
